## 第2章 变量和基本类型

### 基本内置类型

------

基本内置类型包括算术类型(arithmetic type)和空类型(void)

**算数类型**

| 类型         |   最小尺寸   |   含义     |
|:------------ |:---------------- |:---------------- |
| bool         | 未定义   | 布尔类型 |
| char         | 8bits    | 字符     |
| wchar_t      | 16bits   | 宽字节   |
| char16_t     |          |          |
| char32_t     |          |          |
| short        |          |          |
| int          |          |          |
| long         |          |          |
| long  long   |          |          |
| float        |          |          |
| double       |          |          |
| long  double |          |          |
